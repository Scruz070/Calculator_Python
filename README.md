# Python Calculator

This project is an interactive calculator built in Python.  
It allows users to perform basic arithmetic operations continuously through a dynamic menu that repeats until the user chooses to exit.

## Features
- Addition
- Subtraction
- Multiplication
- Division (with zero-division validation)
- Interactive menu with exit option
- Error handling for invalid inputs

## Purpose
The goal of this project is to practice:
- Modular programming
- Control flow (`while`, `if/elif/else`)
- Exception handling (`try/except`)
- User interaction via console

## How to Run
To execute the program, run:
```bash
python Calculator.py
